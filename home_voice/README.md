# Home Voice

Lokaler LLM-Sprachassistent für Home Assistant — Phi-4-mini via llama.cpp,
OpenAI-kompatibel, komplett offline. Mit zweistufigem Gedächtnis
(Ambient-Kontext + optionaler Brain-Recall). Ausführliche Doku: [DOCS.md](DOCS.md).

- Kleine, schnelle Modelle (Phi-4-mini / Qwen2.5 3B / Gemma 3 1B)
- OpenAI-kompatibler Endpunkt für HA Assist
- Deutsche Sprachausgabe über das separate Piper-Add-on (siehe DOCS.md)
- Ambient-Kontext (Anwesenheit, Wetter, To-Dos) ohne Latenz-Kosten
- Optionales semantisches Gedächtnis über Brain
- Alles lokal — keine Cloud, keine Kosten
